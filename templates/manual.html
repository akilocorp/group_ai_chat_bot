ACTR Manual

Group chat for research. Humans join from a survey or the Dashboard. Optional AI personas reply in the room.

Quick start

  1. Open Admin. Set group size, bots, and personas. Click Create Session.
  2. Copy the Qualtrics HTML (if Qualtrics is On) or share the session on the Dashboard.
  3. Participants chat. You watch in Monitor and export CSVs from Export.

Example: 2 humans + 1 bot → set Group size to 2, add one persona, set participant names (e.g. c,d) so they do not clash with the bot name.

Admin fields

Short labels on the Admin page map to these settings.

Session modes

Controls which personas react to each human message.

  • Mode 1 — all bots: Every persona replies to every message.
  • Mode 2 — one bot: The system picks one persona per message.
  • Mode 3 — @mention: A persona replies only when its name is in the message.

Persona modes

Controls when one persona speaks. All modes use delay + typing time.

  • Mode 1 — fixed delay: Always replies after your delay setting.
  • Mode 2 — random delay: Always replies; adds 0.5–2.5 s jitter.
  • Mode 3 — random skip: May skip each message (skip rate). If not skipped, delay + jitter + typing.
  • Mode 4 — context chance: AI estimates reply probability from context, then one random draw.

Qualtrics

  1. In Admin, turn Qualtrics On and create a session. Copy the HTML block.
  2. In Survey Flow → Embedded Data, add fields from the table Admin shows (at least session_id and participant_id in the iframe URL; plus condition if Condition is On; plus transcript and chat_status as outputs).
  3. Paste the HTML into a Text/Graphic → HTML question. Use the full inline block (script + iframe), not only an external script tag.
  4. Test with a real anonymous link. Preview may not save transcript.

When data is saved

  • Chat runs full duration → completed_full
  • Participant clicks Next early → left_early or no_messages
  • Close or refresh → same as Next

Transcript text is capped at about 240,000 characters for Qualtrics.

Production server URL is usually https://group.xjhuang.com. Qualtrics cannot load localhost.

Dashboard

  • Sessions — list sessions, join a queue for testing, open settings.
  • Monitor — read-only live view of active rooms (refreshes every few seconds).
  • Export — download activity log or chat history as CSV.

The top metric AI bots is On if any session has bots enabled.

Chat UI shows plain text with a small avatar (👤 or 🤖). There are no chat bubbles.

FAQ

How do I watch without joining?

Use Dashboard → Monitor.

Can I change a session after creating it?

Use Dashboard → Sessions → open a session → Modify (if the survey window is still open).

Where is data stored?

Local JSON under db/ and activity_logs/, or MongoDB if MONGO_URL is set.

How do I start over?

Create a new session. Old data stays on disk.
